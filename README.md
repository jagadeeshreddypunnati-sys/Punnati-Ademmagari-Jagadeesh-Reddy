# Expense Tracker (Vanilla JS + LocalStorage)

A simple, responsive Expense Tracker built with HTML, CSS and JavaScript.  
Stores transactions (income & expenses) in the browser using `localStorage` â€” no backend required. Perfect as a mini project to show on GitHub.

## Features
- Add transactions with description, amount and type (income / expense).
- Edit and delete transactions.
- Calculates and displays: total balance, total income, total expenses.
- Transactions persist across browser sessions via `localStorage`.
- Responsive layout that works on mobile and desktop.
- Simple, easy-to-read code (great for beginners).

## Demo
Open `index.html` in your browser (double-click or use a local server).
